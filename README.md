# Grokking Algorithms — Python

Python implementations of the algorithms and exercises from the book **Grokking Algorithms** by Aditya Bhargava.

> A companion repository to [data-structures-and-algorithms](https://github.com/iamluhmrt/data-structures-and-algorithms), which addresses similar topics in Java.

## Topics

| Chapter | Topic                                           |
| ------- | ----------------------------------------------- |
| 1       | [Binary Search](./binary-search/)               |
| 2       | [Selection Sort](./selection-sort/)             |
| 3       | [Recursion](./recursion/)                       |
| 4       | [Quicksort](./quicksort/)                       |
| 5       | [Hash Tables](./hash-tables/)                   |
| 6       | [Breadth-First Search](./breadth-first-search/) |
| 7       | [Dijkstra's Algorithm](./dijkstra/)             |
| 8       | [Greedy Algorithms](./greedy-algorithms/)       |
| 9       | [Dynamic Programming](./dynamic-programming/)   |
| 10      | [K-Nearest Neighbors](./knn/)                   |

## Stack

- Python 3.x
- No external dependencies — standard library only

## Running

Each chapter is self-contained. Navigate to any folder and run the file directly:

```bash
cd binary-search
python binary_search.py
```
